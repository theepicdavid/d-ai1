 🤖 D-AI (David-AI)

D-AI is a fast, modern AI chat assistant powered by Groq and LLaMA 3.1.  
It is designed to deliver quick, natural, and helpful conversations through a clean web interface.

Built by **David Does Tech**.

🌐 Live Website: https://d-ai.rf.gd  
👨‍💻 Developer: https://daviddoestech.rf.gd  



## 🚀 Features

- ⚡ Ultra-fast AI responses (Groq-powered)
- 💬 Clean chat UI
- 🧠 Natural, human-like conversation style
- 🌐 Simple web-based interface
- 📱 Mobile-friendly design
- 🔒 Secure API handling via backend



## 🛠️ Tech Stack

- Node.js
- Express.js
- Axios
- Groq API (LLaMA 3.1)
- HTML / CSS / JavaScript
